# CopilotStudio_PromptFiltrodeConteudo
Desafio para a criação de conteúdo utilizando prompt, e criação de filtro de conteúdo

Explorar o Microsoft Copilot
1.1 Abrir o Copilot

Acesse: https://copilot.microsoft.com

Ou, se estiver usando Windows 11, abra pelo ícone do Copilot na barra lateral.

1.2 Testar geração de conteúdo

Experimente:

Criar textos

Criar listas

Reescrever conteúdos

Gerar títulos

Explicar conceitos

Criar imagens (se habilitado)

Explorar o Azure OpenAI
2.1 Acessar o Azure AI Foundry

(antigo Azure OpenAI Studio)

Acesse: https://ai.azure.com

Faça login com a sua conta Azure.

2.2 Criar um projeto

Clique em Projects

Clique em New Project

Nome: lab-openai-dio

Selecione um recurso (Azure AI Services) existente ou crie um.

2.3 Testar um modelo generativo

No menu esquerdo:

Clique em Playground → Chat

Escolha um modelo (GPT-4o, GPT-4 Turbo etc.)

Envie prompts e analise as respostas
